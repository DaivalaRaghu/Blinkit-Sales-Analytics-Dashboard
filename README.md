# Blinkit Sales Analytics Dashboard

## Project Overview
This project presents an interactive sales analytics dashboard built using Microsoft Excel. The dashboard analyzes Blinkit sales data to identify revenue trends, top-performing product categories, and outlet performance.

## Objectives
- Analyze sales data to identify revenue patterns
- Determine top-performing product categories
- Evaluate sales performance by outlet size and location
- Build an interactive dashboard for better data visualization

## Tools & Technologies
- Microsoft Excel
- Pivot Tables
- Power Query
- Slicers
- Charts

## Key Insights
- Fruits & Vegetables contributed **36% of the total revenue**, making it the highest-performing category.
- Medium-sized outlets generated the highest sales.
- Tier 3 locations showed strong customer demand.

## Features
- Interactive dashboard using slicers
- Category-wise revenue analysis
- Sales comparison by outlet size
- Visual charts for better insights

## Dataset
Blinkit sales dataset used for analyzing product categories, outlet types, and revenue distribution.
